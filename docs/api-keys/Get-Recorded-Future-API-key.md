1. Log in to your Recorded Future account at [https://app.recordedfuture.com](https://app.recordedfuture.com).
2. Go to **Settings** → **API Access**.
3. Click **Create new token**, select the **Threat Intelligence** scope, and confirm.
4. Copy the generated token.

Set the `RECORDED_FUTURE_API_KEY` environment variable in your `.env` file or deployment environment.

!!! info
    For more details, refer to the [official Recorded Future API Access documentation](https://support.recordedfuture.com/hc/en-us/articles/115002818507-API-Access).
